# Быстрый старт

## Шаг 1: Установка зависимостей

```bash
npm run install-all
```

Эта команда установит все зависимости для сервера и клиента.

## Шаг 2: Настройка переменных окружения

Создайте файл `.env` в корне проекта:

```env
PORT=5000
JWT_SECRET=your-secret-key-change-in-production
TELEMOST_API_URL=https://telemost.yandex.ru/api/v1
TELEMOST_API_KEY=your-telemost-api-key-here
```

**Примечание:** Если у вас нет API ключа Яндекс Телемост, приложение будет работать в режиме разработки с мок-данными.

## Шаг 3: Создание тестового пользователя (опционально)

```bash
npm run create-test-user
```

Это создаст тестового пользователя:
- Email: `test@example.com`
- Password: `test123`

## Шаг 4: Запуск приложения

```bash
npm run dev
```

Это запустит одновременно сервер (порт 5000) и клиент (порт 3000).

## Шаг 5: Откройте браузер

Перейдите на http://localhost:3000

## Использование

1. **Регистрация/Вход:**
   - Нажмите "Войти" в правом верхнем углу
   - Зарегистрируйтесь или войдите с существующим аккаунтом
   - Или используйте тестового пользователя: `test@example.com` / `test123`

2. **Создание комнаты:**
   - Нажмите "+ Создать комнату"
   - Заполните форму (название обязательно)
   - Выберите уровень доступа
   - Комната автоматически создастся в Яндекс Телемост

3. **Присоединение к комнате:**
   - Нажмите "Присоединиться" на карточке комнаты
   - Откроется новая вкладка с видеовстречей в Яндекс Телемост

## Отдельный запуск сервера и клиента

**Только сервер:**
```bash
npm run server
```

**Только клиент:**
```bash
npm run client
```

## Устранение проблем

### База данных не создается
Убедитесь, что у приложения есть права на запись в папку `server/database/`

### Ошибки при создании комнаты
- Проверьте, что API ключ Яндекс Телемост правильно указан в `.env`
- Если ключа нет, приложение будет использовать мок-данные

### Порт уже занят
Измените порт в `.env` файле или остановите процесс, использующий порт 5000/3000
